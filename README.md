## CFG to PDA Converter

Entering a PDA:
* MUST be in Greibach Normal Form for code to run correctly
* Each terminal should be a different letter (no subscripts)
* Example format: S->aAB|bCB|SS
    * __NO__ __spaces__
        * Allowed: aAB|bCB
        * Not allowed: aAB | bCB
